# Photo placement guide

Your website tries multiple filename patterns so photos still show even if extension/naming differs.

## Preferred filenames
Place your photos in this folder using:
- betra-01.jpg
- betra-02.jpg
- betra-03.jpg
- betra-04.jpg
- betra-05.jpg
- betra-06.jpg
- betra-07.jpg
- betra-08.jpg
- betra-09.jpg
- betra-10.jpg
- betra-11.jpg
- betra-12.jpg

## Also supported automatically
- Extensions: `.jpg`, `.jpeg`, `.png`, `.webp` (upper/lowercase)
- Legacy names from earlier site versions: `ava-01` ... `ava-12`

If no match is found, the site keeps showing `fallback.svg` so you never see broken image icons.
